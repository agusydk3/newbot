## 📝 Cloning This Repo
```bash
> git clone https://github.com/SlavyanDesu/simple-wa-bot
> cd simple-wa-bot
```

## 🔍 Installing The Dependencies
```bash
> npm install
```

## 🆗 Running The Bot
```bash
> npm start
```
After that scan the QR code and wait until it authenticated.
